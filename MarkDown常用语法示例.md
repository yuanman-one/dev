## 列表

### 有序列表

- 使用`1. xxx`，`2. xxx`表示依次类推

 1. 第一列
 2. 第二列
 3. 第三列
 4. 第四列

#### 语法示例

  ```md
  1. 第一列
  2. 第二列
  3. 第三列
  4. 第四列
  ```

### 无序列表

**使用`- xxx`，`* xxx`，`+ xxx`表示**

- 是

- 点

- 我

- 打

#### 语法示例

- $\color{red}{注意}$使用`-` `*` `+`效果都是无序列表

  ```md

  - 是
  - 点
  - 我
  - 打
   ```
